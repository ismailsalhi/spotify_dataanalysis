# Spotify User Behavior Analysis

## About the Project
This is a Data Analysis project for our group at Rennes School of Business. We are analyzing Spotify user data to find patterns in listening habits and subscription choices.

## Folders and Organization
* **data/**: Contains the raw CSV files for users, listening habits, and plans.
* **notebooks/**: Contains the `SPOTIFY_DATATIL.ipynb` file with our Python code.
* **reports/**: Contains the final `DATATIL REPORT.pdf`.

## Tools Used
* Python (Pandas, Matplotlib)
* Jupyter Notebooks
* GitHub# spotify_dataanalysis
