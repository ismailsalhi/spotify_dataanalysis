# Spotify User Behavior & Subscription Analysis

## About the Project
This project was developed for the **Programming for Data Analysis** course at **Rennes School of Business**. Our goal is to analyze how different factors such as age, gender, and daily usage habits influence a user's choice between "Free" and "Premium" Spotify subscriptions.

By using Python and NumPy, we process raw user data to identify trends in streaming behavior and provide a data-driven overview of Spotify's user base in 2024.

## The Dataset
We utilized three main datasets for this analysis:
* **`spotify_users.csv`**: Contains demographic details like age and gender.
* **`spotify_listening_daily.csv`**: Tracks daily active minutes and song counts.
* **`spotify_plans.csv`**: Details regarding subscription tiers and pricing.

## Folders and Organization
To maintain a professional structure, the repository is organized as follows:
* `/data`: Contains all raw CSV files.
* `/notebooks`: Contains the primary analysis file `SPOTIFY_DATATIL.ipynb`.
* `/reports`: Contains the comprehensive `DATATIL REPORT.pdf`.

## Methodology (EDA)
Our Exploratory Data Analysis (EDA) followed these steps:
1. **Data Cleaning**: Handling missing values and ensuring data types are correct for analysis.
2. **Statistical Summaries**: Using **NumPy** to calculate the mean and median listening times across different demographics.
3. **Correlation Analysis**: Investigating the link between "Daily Hours Streamed" and "Plan Type".
